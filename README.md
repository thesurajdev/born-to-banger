## 🚀 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/thesurajdev/born-to-banger.git
   ```

2. Open the folder:

   ```bash
   cd born-to-banger
   ```

3. Run in browser:

   * Open `index.html` directly
   * Or use Live Server (VS Code recommended)
